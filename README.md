# Mitzy Travel and Tours Website

Live site: https://mitzytravelandtours.com

A full-stack travel agency website with a custom content management system (CMS) for managing tours, media, and website content.
Built with PHP, JSON, and custom admin panel.

## 🧠 System Highlights

- 📩 Inquiry system that captures user submissions and routes them to business email workflows
- 📈 Supports real-world business operations (tour inquiries, lead generation, and customer communication)
- 🧾 Structured data handling and logging for managing content updates and system activity


## ⚙️ Setup (Local Development)
1. Install XAMPP or any PHP environment
2. Place the project folder in `htdocs/`
3. Start Apache
4. Open in browser:
   http://localhost/your-folder

## ⚠️ Notes

- The live version is deployed on a hosted server (Hostinger)
- Local setup may require additional configuration for full functionality
- Admin access is required for CMS features
- Credentials can be configured in the backend/database

## ⚡ Features
- 🌍 Dynamic tour listing system
- 🛠️ Custom admin panel (CMS)
- 🖼️ Image upload and media handling
- 🔄 Dynamic content updates via CMS
- 📱 Responsive UI

## ⚡ Advanced Features

### Content & UI
- Accordion-based itinerary system with embedded images
- Flyer preview for tour packages
- Drag & Drop arrangement for packages

### System & Data Handling
- JSON import/export functionality
- Structured data management for tours and content

### User Interaction
- Search and filtering system for tour packages and visas (editable in the CMS)
